AngularJS

AngularJS features

  • Automatic bidirectional binding
  • Controllers
  • DI
  • Readable customised HTML thanks to directives. AngularJS is a tool to build your tools (think DSL)
  • Controller separated from view (think MVC) = testable
  • One Page App, with deep linking (think http://app/#/bookmarkable/url
  • Services on top of XHR
  • Tool, not framework. You can have AngularJS in tiny part of your page, or everywhere
We'll not talk about: E2E - AngularJS integration tests
Bidirectional binding. Change the model - the display is changed too. Angular event loop.
